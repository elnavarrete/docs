# CallBacks

``` js

let getUserById = ( id, callback ) => {

    let user = {
        name: 'Erik',
        id         // se pone un solo ID xq en ES6 no es necesario poner el mismo nombre de parametro
    };

    if (id === 20) {    // Con esto simulo un error
        callback(`El usuario con id ${id}, no existe en la BD`);
    } else {
        callback(null, user);
    }


};

```
